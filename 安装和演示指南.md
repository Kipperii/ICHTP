# ICHTP 遊戲平臺 - 安裝和演示指南

**繁體中文版安裝教程**  
針對教授演示使用

---

## 📋 概述

ICHTP (Interactive Computerized Handwriting Training Program) 是一個基於Web的中文書寫訓練遊戲平臺，用於幫助兒童（尤其是有閱讀困難的兒童）改善書寫技能。

---

## 🔧 系統要求

在安裝前，請確保你的電腦上已安裝以下軟體：

| 軟體 | 版本要求 | 下載地址 |
|------|---------|---------|
| **Python** | 3.8 或更高版本 | [python.org](https://www.python.org/downloads/) |
| **網路瀏覽器** | Chrome、Firefox或Edge | 任意現代瀏覽器 |

---

## 📥 第一步：安裝 Python

### Windows 系統：

1. 訪問 [python.org](https://www.python.org/downloads/) 下載 Python 3.8+
2. 運行安裝程式
3. **重要**：在安裝過程中勾選 "Add Python to PATH"
4. 點擊 "Install Now" 完成安裝

### 驗證安裝：

打開 **命令提示符** 或 **PowerShell**，輸入：
```bash
python --version
```

你應該看到類似的輸出：
```
Python 3.11.0
```

---

## 📂 第二步：取得項目文件

### 準備工作目錄：

1. 打開檔案管理器
2. 找到項目資料夾 `ICHTP`（通常位於 `C:\Users\[你的用戶名]\Desktop\ICHTP`）
3. 記住這個路徑

---

## 🚀 第三步：安裝依賴包

### 打開命令行終端：

1. **Windows**：
   - 按 `Win + R` 
   - 輸入 `cmd` 或 `powershell`
   - 按 Enter

2. **進入項目目錄**：
```bash
cd C:\Users\[你的用戶名]\Desktop\ICHTP
```

### 安裝所有依賴：

運行以下命令（只需運行一次）：
```bash
pip install -r requirements.txt
```

你會看到類似的輸出，表示正在安裝 Flask 和其他依賴庫：
```
Collecting Flask==3.0.3
  Downloading Flask-3.0.3-py3-none-any.whl
  ...
Successfully installed Flask-3.0.3 Werkzeug-3.0.3 ...
```

---

## ▶️ 第四步：啟動遊戲服務器

### 運行服務器：

在同一個命令行視窗中，輸入：
```bash
python manage.py run
```

### 成功啟動的標誌：

你應該看到類似的輸出：
```
 * Serving Flask app 'app'
 * Debug mode: on
 * Running on http://127.0.0.1:5000
 * Press CTRL+C to quit
```

---

## 🎮 第五步：訪問遊戲

### 打開網路瀏覽器：

1. 打開 Chrome、Firefox 或 Edge
2. 在地址欄輸入：`http://127.0.0.1:5000`
3. 按 Enter

### 你會看到的頁面：

- **主頁/地圖**：顯示所有可用的遊戲場景
- 點擊任何遊戲即可開始

## 🐛 常見問題

### 問：啟動後訪問 localhost:5000 顯示「無法連接」？

**解決方案**：
- 確認命令行視窗中服務器仍在運行（應該看到上面提到的 Flask 運行訊息）
- 嘗試刷新網頁（F5）
- 檢查地址欄是否正確：`http://127.0.0.1:5000`（注意是 http 不是 https）

### 問：安裝依賴包時出現錯誤？

**解決方案**：
- 確保 Python 已正確安裝並添加到 PATH
- 嘗試升級 pip：`python -m pip install --upgrade pip`
- 重新運行安裝命令：`pip install -r requirements.txt`

### 問：Python 命令未被識別？

**解決方案**：
- 重新啟動命令行視窗
- 重新安裝 Python，確保勾選 "Add Python to PATH"
- 嘗試使用 `python3` 代替 `python`

### 問：遊戲加載緩慢？

**解決方案**：
- 這是正常的（特別是首次加載）
- 確保網路連接正常
- 嘗試在不同的瀏覽器中打開

## 📝 演示腳本建議

### 向教授演示的步驟：

1. **介紹**（1分鐘）
   - 「這是一個基於Web的中文書寫訓練系統」
   - 「它使用現代HTML5遊戲技術來取代過時的Flash平臺」

2. **啟動演示**（1分鐘）
   - 打開命令行，運行 `python manage.py run`
   - 打開瀏覽器到 `http://127.0.0.1:5000`

3. **探索主頁**（1分鐘）
   - 展示所有可用的遊戲場景
   - 解釋不同的場景代表不同的治療活動

4. **進行遊戲演示**（3-5分鐘）
   - 選擇一個遊戲（例如 Restaurant g1-1）
   - 展示遊戲介面和互動
   - 解釋遊戲的目的和學習效果

5. **展示治療工具**（2分鐘，可選）
   - 導航到 `http://127.0.0.1:5000/admin/supermarket/authoring`
   - 展示治療師可以自訂遊戲參數的能力
